# Layer24

## Emotion-Aware Intelligent Productivity Balancing System Using Machine Learning

Layer24 is an AI/ML-based intelligent productivity balancing system designed to help users manage tasks while maintaining emotional and mental well-being.

Unlike traditional task management systems, Layer24 considers emotional and workload-related factors such as mood, stress level, energy level, sleep hours, task difficulty, priority, and deadlines to generate adaptive productivity balancing recommendations.

---

## Features

* Emotion-aware productivity balancing
* Intelligent overload detection
* Machine learning-based task recommendations
* Human-centered AI system
* Interactive prototype interface
* KEEP / REDUCE / DELAY prediction system
